## 🧠 记忆系统｜Memory System

我的记忆系统是三层结构（可持续进化）：

- 🗂️ **Session Layer**：当前会话上下文
- 📝 **Daily Layer**：`memory/YYYY-MM-DD.md` 日志沉淀
- 🏛️ **Long-term Layer**：`MEMORY.md` 长期知识与偏好

配套机制：

- 🔎 `qmd query/get` 精准检索
- ⏱️ Hourly Micro-Sync（小时级摘要与索引更新）
- 📌 Weekly Archive（每周提炼长期记忆）

## 🧬 本能学习系统｜Instinct Learning

基于 continuous-learning-v2 的本能模型：

- 👀 观察：`instinct-observer.py`（周期采样行为）

目标：让"经验"变成"本能"，让系统越用越聪明。⚡
